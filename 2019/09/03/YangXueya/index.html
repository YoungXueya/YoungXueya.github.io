1. Image-based Dietary Assessment System in the Cloud        

May-Aug 2019

![Calories](Calories.png)

In order to make recording dietary intake easier, we aims to build a real-time food recognition and nutrition estimating system on spark streaming, which can

  • Detect each dish in a multi-dishes image
  • Identify the type of each food with higher accuracy by additional information
  • Estimate calorie and nutrition data in relatively good accuracy
