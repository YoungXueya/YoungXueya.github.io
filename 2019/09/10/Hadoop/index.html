Hadoop

2019-09-10

Introduction

The Apache Hadoop software library is a framework that allows for the distributed processing of large data sets across clusters of computers using simple programming models.

Hadoop 2.x takes Hadoop beyond batch application: Allow for multiple ways to interact with data stored in HDFS: (1) batch with MapReduce(MR), (2) streaming with Spark/Storm, (3) interactive SQL with Spark SQL, Hive.

Apache Hadoop contains the following modules:

  1. Hadoop Common- contains libraries and utilities needed by other Hadoop modules.
  2. Hadoop Distributed File System(HDFS): A distributed system for storing data
  3. Hadoop MapReduce: A programming model for large scale data processing
  4. Hadoop YARN: resource-management and task scheduling.

Hadoop 1.x: Key Components

  1. Job Tracker@master node

    Keeps track of all MapReduce jobs that are running on various nodes.

  2. Name Node@master node

    Store metadata: file/chuck namespaces, file-to chunk mapping, location of each chunk’s replicas.

  3. Task Tracker@each slave node

    A slave to Job Tracker. Lauches child processes(JVMs) to execute the map or reduce tasks.

  4. Data Node@each slave node

    A slave to the Name Node. Stores data on the local file system.
